# PseudorangeMethod
This project contains an algorithm to solve the issue of founding receiver coords by pseudorange method

В файле pseudorangeMethod.m хранится код функции для расчёта координаты навигационного приёмника. 

В файле test.m хранится код, который демонстрирует использование функции pseudorangeMethod(). Для корректной работы файлы test.m и pseudorangeMethod.m должны лежать в одной директоррии.

Для корректной работы функции pseudorangeMethod(), файл с кодом функции должен лежать в одной папке со скриптом, в котором вызывается эта функция, либо же файл должен лежать в каталоге, который входит в список известных Matlab каталогов.
